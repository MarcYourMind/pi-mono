# LinkedIn Hiring Search Workflow

## Agents
### LinkedIn Access Agent
Handle authentication, API connections, and query capabilities on LinkedIn.

### Search Execution Agent
Run automated searches for tech businesses, focusing on startups or companies with recent funding, seeking growth.

### Profile Identification Agent
Identify businesses actively hiring (based on their posted job listings or growth signals).

### Data Extraction Agent
Extract key details (company name, job listings, location, profile links) for each hiring business.

### Data Filtering Agent
Apply filters to ensure the businesses match your target criteria (e.g., funding stage, size, growth indicators).

### Document Compilation Agent
Compile the filtered results into a structured document (e.g., a spreadsheet or PDF) with company names, job listings, and links.

### Link Validation Agent
Verify that all profile links are active and correctly formatted.

### Priority Flag Agent
Highlight high-potential businesses for direct outreach or further personalized follow-up.

### Logging Agent
Log all searches, results, and any issues for transparency and refinement.
